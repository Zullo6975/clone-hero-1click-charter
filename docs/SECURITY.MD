## Reporting a Vulnerability

We take the security of our software seriously. If you have found a vulnerability, please do not open a public issue.

1.  **Report:** Please email details to [INSERT YOUR EMAIL HERE] or report via GitHub's "Security" tab if enabled.
2.  **Response:** We will acknowledge your report within 48 hours.
3.  **Resolution:** We will work to verify and patch the vulnerability as soon as possible.
